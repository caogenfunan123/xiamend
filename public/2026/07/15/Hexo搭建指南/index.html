使用 Hexo 搭建个人博客完全指南

2026年07月15日 · 技术 · 标签：Hexo, 教程, 博客

Hexo 是一个快速、简洁且高效的博客框架，支持 Markdown 语法，可以轻松生成静态网站。本文将详细介绍如何从零开始搭建自己的 Hexo 博客。

一、环境准备

首先需要安装 Node.js 和 Git。可以通过以下命令检查是否已安装：

node -v
git --version

安装 Hexo CLI：

npm install -g hexo-cli

二、初始化博客

在你想存放博客的目录下执行：

hexo init blog
cd blog
npm install

三、创建文章

使用以下命令创建新文章：

hexo new "我的第一篇文章"

这会在 source/_posts/ 目录下生成一个 Markdown 文件，你可以用任何编辑器编辑它。

四、本地预览

hexo server

打开浏览器访问 http://localhost:4000 即可预览。

五、生成静态文件

hexo generate

这会在 public/ 目录下生成所有静态 HTML 文件。

六、部署到 GitHub Pages

修改 _config.yml 中的 deploy 配置：

deploy:
  type: git
  repo: https://github.com/用户名/用户名.github.io.git
  branch: main

安装部署插件：

npm install hexo-deployer-git --save

执行部署：

hexo deploy

七、部署到 Cloudflare Pages

  1. 将代码推送到 GitHub 仓库
  2. 在 Cloudflare Dashboard 中创建新的 Pages 项目
  3. 连接 GitHub 仓库
  4. 构建命令填写：npm install && npx hexo generate
  5. 输出目录填写：public
  6. 点击保存并部署

八、更换主题

Hexo 有很多优秀主题，你可以从 官方主题库 选择，也可以自己定制。将主题放到 themes/ 目录下，修改 _config.yml 中的 theme 字段即可。

建议先从简单主题开始，熟悉后逐步定制。

总结

Hexo 搭建博客非常简单，只需要几步就可以拥有一个属于自己的博客。希望这篇文章对你有帮助！
